# Aliexpress Scraper

Scrapping products from Aliexpress

## Methods

- Find items by category

## How to use?

```js
import { getItemsByCategory } from 'aliexpress-scraper';

const items = await getItemsByCategory({
	category: 'kitchen-dining-bar',
	page: 1,
	minPrice: 1,
	maxPrice: 2000,
});
```
